# html-mini-project
Mini Project Report

Title: Quiz on HTML

1. Introduction

This project is an HTML-only online quiz developed as a mini project. It is designed to test users' basic knowledge of HTML concepts through multiple-choice questions.


2. Objective

The main objectives of this project are:

To understand the structure of HTML documents

To learn how to use forms in HTML

To implement MCQ-based quizzes

To practice linking multiple web pages

3. Scope of the Project

This project is limited to HTML only. No CSS or JavaScript has been used. The quiz is static and does not calculate scores automatically.

4. Tools & Technologies Used

HTML5

Web Browser (Chrome, Edge, Firefox, etc.)

GitHub for hosting

5. System Requirements

Hardware Requirements:

Computer or Mobile Device

Minimum 2GB RAM

Software Requirements:

Web Browser

Text Editor (VS Code, Notepad, etc.)

6. Project Modules

a) Home Page

Displays Instructions

Contains start quiz 

Opens in a new tab

b) Quiz Page

Displays questions

Shows 4 options per question

Uses radio buttons

Opens in a new tab

c) Answer Key Page

Displays correct answers

Opens in a new tab


7. Features

User-friendly

Beginner-level

Static quiz

Easy navigation

Simple design



8. Limitations

No automatic score calculation

No backend

No user authentication

No database



9. Future Enhancements

Add CSS for better design

Add JavaScript for score calculation

Store results



10. Conclusion

This HTML quiz project helped in understanding the basics of web development using only HTML. It improved knowledge of forms, page linking, and webpage structure.
